# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/VinewZ/VO-Filmpire.git
```

```bash
# Install Dependencies
$ yarn install

# Run Aplication
$ yarn run dev
```
Go to http://localhost:3000/ to see the result.
